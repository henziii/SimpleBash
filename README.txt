В этом проекте были разработанны базовые утилиты Bash (cat, grep) для работы с текстовыми файлами. Разработанные утилиты полностью повторяют поведение оригинальных cat и grep. Код написан на языке программирования Си и находится в директории src. Для быстрой сборки проекта были написаны Makefile. Чтобы сбилдить проект нужно прописать в консоли "make all", после чего создастся исполняемый файл s21_cat (или s21_grep, в зависимости от утилиты). Для точной проверки работоспособности кода были написаны Unit-тесты. Тесты для каждой утилиты находятся в соответствующей названию утилиты папке в файле test.py. Для проверки тестами прописываем в консоль "python3 test.py".

===================================================================================================

Для теста утилиты cat вручную прописываем в консоль "./s21_cat [OPTIONS] [FILE_NAME]..."

Поддерживаемые опции для утилиты cat([OPTIONS]):
-b | нумерует только непустые строки.
-e (предполагает и -v) | также отображает символы конца строки как $.
-n | нумерует все выходные строки.
-s | сжимает несколько смежных пустых строк.
-t (предполагает и -v) | также отображает табы как ^I.
-v | отображает все непечатаемые символы.

===================================================================================================

Для теста утилиты grep вручную прописываем в консоль "./s21_grep [OPTIONS] template [FILE_NAME]..."

Поддерживаемые опции для утилиты grep([OPTIONS]):
-e | Шаблон.
-i | Игнорирует различия регистра.
-v | Инвертирует смысл поиска соответствий.
-c | Выводит только количество совпадающих строк.
-l | Выводит только совпадающие файлы.
-n | Предваряет каждую строку вывода номером строки из файла ввода.
-h | Выводит совпадающие строки, не предваряя их именами файлов.
-s | Подавляет сообщения об ошибках о несуществующих или не читаемых файлах.
-f file | Получает регулярные выражения из файла.
-o | Печатает только совпадающие (не пустые) части совпавшей строки.
